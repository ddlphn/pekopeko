# Construction d'une phrase

Sujet + Particule | Complément + Particule | Verbe (+ きだ)

## Phrase nominale

### Affirmative は + です

私 **は** デルフィン **です** / Je m'appelle Delphine

日本 **は** 国 **です** / Le Japon est un pays

### Interrogative は + か

日本 **は** 国 **ですか** / Le Japon est un pays ?

## Phrase verbale

### Affirmative ます
